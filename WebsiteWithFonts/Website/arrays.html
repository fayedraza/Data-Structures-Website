Declaring Values

1D arrays are just an array with 1 dimension, meaning the area is 1 * n and n is the size which we declared.

In the image below, we declared an array in Java with a fixed length.
Also, every array has default values once initialized, the default values depend on the type, such as, String, int, and etc.

So how can we declare a value?
We just need to set the value using an index which is an access to that value in the array.
The small values at the bottom of the array are indexes. We can set it to any value using that.
In addition, the range of indexes is 0 to the length of an array minus 1. Anything out of that range makes it invalid.

What if we set the value to an index that is out of bounds?
An out of bounds exception is thrown WHICH WE DO NOT WANT.
